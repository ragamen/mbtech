1. Descripción del Servicio

MBTech proporciona una solución tecnológica de software que actúa como pasarela o panel de gestión para el API Oficial de WhatsApp Cloud de Meta. El Servicio permite la conexión, el monitoreo del webhook de eventos y el envío de plantillas autorizadas o mensajes conversacionales a contactos del cliente.

2. Requisitos y Cumplimiento con Meta

Para utilizar nuestro Servicio, debes disponer de una cuenta de WhatsApp Business API vinculada a tu propio Meta Business Manager. Es tu responsabilidad exclusiva cumplir estrictamente con:

  • Las Condiciones comerciales de WhatsApp.
  • Las Políticas de mensajería de WhatsApp Business (prohibición de spam, distribución de contenido fraudulento, de juego, etc.).
  • Tener el consentimiento expreso (Opt-in) de tus contactos finales para enviarles mensajes informativos o promocionales.
